# phpEcommerce
This is an Ecommerce site made from scratch.The eccomerce site has 3 divisions 1st is the main page then one page  for each product and finally a cart page. Login functionality has been added for the e-commerce part. 3 database have to be made in order for the project to work with user, products, and cart. As for CMS side admin can view users but password is encrypted , add products delete products delete users etc.
